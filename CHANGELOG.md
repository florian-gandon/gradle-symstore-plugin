Plugin changelog
====================

0.2
-------

* Added comment option. The default value for the comment is the user and the machine.
* Added version option. The default value is the project version.
* Added verbose logging from symstore when Gradle debug level is set


0.1
-------

* Added compress option, be default false.
* Added lock for share folder
* Added "addSymbol" task to add symbols to a store.
